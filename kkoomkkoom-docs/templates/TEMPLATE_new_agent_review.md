# 신규 에이전트 설계 검토 체크리스트

---
- **대상 에이전트**: (에이전트명 기입)
- **소속 팀**: (팀명 기입)
- **설계 담당**: (담당자 기입)
- **검토 시작일**: YYYY-MM-DD
- **버전**: v1.0
- **상태**: 검토 중 | 검토 완료
---

## 사용 가이드

**언제 사용하는가**: 신규 에이전트를 설계할 때, 설계서 초안 작성 전 또는 작성 후 각 팀장에게 리뷰를 요청할 때 사용한다.

**사용 방법**:
1. 이 템플릿을 복사하여 `[팀폴더]/tasks/YYYY-MM/` 에 저장
2. 상단 메타 정보를 기입
3. 해당 팀장에게 이 문서 + 설계 초안을 함께 전달
4. 각 팀장이 자기 섹션의 체크리스트를 검토하고 의견을 기입
5. 전체 피드백을 수합하여 설계서에 반영

**전체 팀장에게 보낼 필요 없음**: 해당 에이전트와 관련된 팀장에게만 선택적으로 요청한다.
