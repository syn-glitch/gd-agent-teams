# 🐰 주디(Judy) AI 비서 챗봇 통합 가이드 (v2.9.0)

본 문서는 '공도 업무 관리' 시스템의 핵심 인터페이스인 **주디 AI 비서 챗봇 (사이드 패널)**의 전체 구성과 기술 사양, 그리고 활용 방법을 정리한 통합 가이드입니다.

---

## 1. 시스템 개요 (System Overview)

주디(Judy)는 사용자의 업무 데이터를 실시간으로 이해하고 답변하는 **RAG(Retrieval-Augmented Generation) 기반 AI 비서**입니다. 구글 앱스 스크립트(GAS) 환경에서 동작하며, 서비스 전체 화면 우측에 사이드 패널 형태로 상시 대기합니다.

### 🌟 핵심 가치
- **데이터 통합**: 산재된 업무 시트, 마크다운 노트, 구글 드라이브 아카이브를 한곳에서 검색.
- **가심비 UX**: Claude v3.5 기반의 친절한 페르소나와 직관적인 사이드바 인터페이스.
- **실시간성**: 1분 단위 캐싱과 최신 데이터 동기화를 통한 즉각적인 답변.

---

## 2. 기술 아키텍처 (Architecture)

### 🧩 구성 요소
- **Frontend**: `judy_workspace.html` (HTML5, Vanilla CSS3, JS)
- **Backend**: `ai_chat.gs` (Google Apps Script)
- **AI Engine**: Claude v3.5 Sonnet (Anthropic API)
- **Data Source**: 
    - Google Sheets (`Tasks`, `Notes`, `Calendar_Events`)
    - Markdown Notes (`Notes_v2` API)
    - Google Drive Archive (`drive_archive.gs` 파서 연동)

### 🛠 주요 기술 스펙
| 항목 | 기술/로직 | 비고 |
|------|-----------|------|
| **UI 레이아웃** | Flexbox 기반 사이드 패널 | 가로폭 조절(Resizer) 기능 포함 |
| **IME 대응** | `isComposing` 감지 로직 | 한글 입력 시 중복 입력 버그 방지 |
| **검색 엔진** | 하이브리드 RAG | 시트 SQL + 텍스트 서치 + 아카이브 파싱 |
| **로딩 상태** | "당근펜" 동적 GIF 인디케이터 | 클로드/OpenAI 스타일의 메시지 하단 배치 |

## 3. 기능 및 사용법 (User Guide)

### 3.1 챗봇 호출 및 인터랙션
- **열기/닫기**: 우측 하단의 🐰 아이콘을 클릭하여 패널을 여닫을 수 있습니다.
- **폭 조절**: 패널 왼쪽 경계선을 드래그하여 채팅창 폭을 자유롭게 조절할 수 있습니다.
- **전송**: 텍스트 입력 후 `Enter` (모바일은 전송 버튼 클릭)로 질문을 보냅니다.

### 3.2 검색 가능한 데이터 범위
주디는 다음 영역을 모두 뒤져서 답변을 생성합니다.
- **업무(Tasks)**: 현재 진행 중이거나 대기 중인 모든 프로젝트 업무.
- **메모(Notes)**: 일반 시트 메모의 제목 및 본문.
- **마크다운 노트**: 사용자가 직접 작성한 상세 마크다운 일지.
- **과거 아카이브**: 구글 드라이브에 보관된 과거 수개월~수년 치의 업무 메모.

### 3.3 출근 브리핑 (v2.9.0 신규)

채팅창에 **"출근"**이라고 입력하면 주디가 AI 호출 없이 **1~2초 내**에 당일 업무와 일정을 요약해줍니다.

**사용법**: 채팅창에 `출근` 입력 후 Enter

**응답 구성**:
- 🌅 **랜덤 인사말** — 매일 다른 인사로 기분 좋은 아침을
- 🔴 **오늘 마감** — 당일 마감인 업무 목록
- 🟡 **진행중** — 현재 진행 중인 업무 (마감일 포함)
- ⚪ **대기** — 아직 시작 전인 업무
- 📅 **오늘 일정** — 캘린더에 등록된 오늘 일정 (시간순)

**권한별 표시 범위**:
| 사용자 유형 | 표시 업무 |
|------------|----------|
| 일반 직원 | 본인 담당 업무만 |
| 관리자 (송용남, 정혜림, 김관수) | 전체 팀원 업무 (담당자 표시) |

> **💡 팁**: 출근 브리핑은 AI API를 사용하지 않아 빠르고, 토큰 비용이 발생하지 않습니다. 매일 아침 습관적으로 사용하세요!

## 4. 실전 활용 사례 (Examples)

### 🔍 사례 1: 특정 프로젝트 업무 확인
> **질문**: "전주교육지원청 관련 업무 중 마감이 제일 가까운 게 뭐야?"
> 
> **AI 답변**: "현재 진행 중인 [전주교육지원청 학교 밖 디지털튜터] 업무가 있으며, 마감일은 2026-03-09입니다. 담당자는 '송용남'님으로 확인됩니다."

### 🔍 사례 2: 과거 아카이브 검색 (넷마블 사례 등)
> **질문**: "예전에 넷마블이랑 미팅했던 내용이 아카이브에 있어?"
> 
> **AI 답변**: "네, 2025년 10월 12일자 아카이브 메모에서 '넷마블 협력 미팅 일지'를 찾았습니다. 주요 내용은 [게임 기반 교육 플랫폼 연동 건]이며, 상세 이력이 구글 문서로 보관되어 있습니다."

### 🔍 사례 3: 마크다운 노트 요약
> **질문**: "어제 쓴 마크다운 노트 내용 요약해줘."
> 
> **AI 답변**: "어제(3월 7일) 작성하신 노트는 'RAG 시스템 개선안'에 대한 내용입니다. 주요 포인트는 1) 캐시 주기 단축, 2) 아카이브 파서 연동, 3) IME 버그 수정 계획 등 3가지입니다."

### 💻 프론트엔드 엔터 키 핸들러 (`judy_workspace.html`)
```javascript
window.handleJudyChatKey = function (e) {
    if (e.isComposing) return; // 한글 IME 중복 트리거 방지 (v2.8.6 패치)
    if (e.key === 'Enter' && !e.shiftKey) {
        e.preventDefault();
        window.sendJudyChat();
    }
};
```
