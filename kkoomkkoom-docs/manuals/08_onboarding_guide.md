# 🚀 신규 팀원 온보딩 가이드

> **최종 업데이트**: 2026-03-09 | **버전**: v1.0

---

## 환영합니다!

공도 업무 관리 시스템 **주디 워크스페이스(Judy Workspace)**에 오신 것을 환영합니다.
이 가이드를 따라하면 5분 안에 시스템 사용을 시작할 수 있습니다.

---

## 1단계: 접속하기

### Slack에서 접속 (권장)

1. Slack에서 **Judy Ops** 봇에게 DM을 보냅니다
2. `노트` 또는 `메모장`이라고 입력합니다
3. 받은 **매직 링크**를 클릭합니다 (10분 내 클릭, 1회용)
4. 주디 워크스페이스가 열립니다

### Slack 명령어로 접속

```
/주디 내업무     → 내 업무 대시보드
/주디 메모장     → 내 노트
```

> 매직 링크는 **10분 후 만료**, **1회 클릭 시 소멸**됩니다.
> 구글 로그인 없이 Slack 인증으로 접속합니다.

---

## 2단계: 워크스페이스 둘러보기

상단 탭으로 기능을 전환합니다:

| 탭 | 기능 | 설명 |
|----|------|------|
| 📝 **내 노트** | 메모·회의록 | 날짜별 메모 작성, AI 요약, 검색 |
| 📊 **내 업무** | 업무 관리 | 업무 등록·상태 변경·마감일 관리 |
| 📋 **칸반** | 칸반 보드 | 드래그&드롭으로 업무 상태 변경 |
| 📅 **달력** | 캘린더 | 마감일 기준 업무 캘린더 뷰 |

---

## 3단계: 주요 기능 사용법

### 메모 작성

1. **내 노트** 탭 클릭
2. 텍스트 입력 후 **저장** 클릭 (또는 `Cmd + Enter`)
3. 마크다운 문법 지원 (제목, 굵게, 리스트 등)
4. 좌측 사이드바에서 날짜별 과거 메모 조회

### 업무 확인

1. **내 업무** 탭 클릭
2. 요약바에서 전체 업무 현황 확인 (진행중/대기/보류/완료)
3. 업무 행을 클릭하면 **상세 모달**이 열림
4. 상태 드롭다운으로 상태 변경 (대기 → 진행중 → 완료)

### 업무 등록

1. **+ 새 업무 등록** 버튼 클릭
2. 프로젝트 선택 (또는 새 프로젝트 생성)
3. 업무 제목, 내용, 마감일 입력
4. **담당자**: 본인이면 그대로, 다른 사람이면 선택 (수락대기 상태로 전달됨)
5. **등록** 클릭

### 업무를 전달받았을 때

다른 팀원이 나에게 업무를 전달하면:
1. Slack DM으로 알림이 옵니다
2. 내 업무 탭에 **🤝 수락대기** 상태로 표시됩니다
3. **✅ 수락** 버튼을 클릭하면 → **진행중**으로 변경되고 업무가 시작됩니다
4. 요청자에게 수락 알림 DM이 자동 발송됩니다

### 참고사항 추가

업무 상세 모달 하단의 **📎 참고사항** 영역에서:
1. **+ 추가** 버튼 클릭
2. 텍스트 또는 링크 입력 후 **등록**
3. 수정(✏️) / 삭제(🗑️) 가능
4. 모든 변경 이력이 자동 보관됩니다

## 4단계: 주디 AI 비서 활용

화면 우측 하단 **🥕 주디** 버튼을 클릭하면 AI 챗봇이 열립니다.

| 입력 예시 | 기능 |
|----------|------|
| `출근` | 오늘의 업무 브리핑 |
| `내 업무 알려줘` | 업무 현황 요약 |
| `지난주 회의 내용` | 메모 검색 (RAG) |
| 자유 질문 | AI 답변 |
