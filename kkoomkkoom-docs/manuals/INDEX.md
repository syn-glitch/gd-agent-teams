## 📝 관리 규칙 (꼼꼼이 가이드)
1. 모든 신규 매뉴얼은 본 인덱스에 등록 후 배포합니다.
2. 매뉴얼 내용 수정 시 버전 번호를 업데이트하고 변경 이력을 남깁니다.
3. 구버전이나 파편화된 매뉴얼은 발견 즉시 본 폴더(`manuals/`)로 통합합니다.
