#### [MINOR/기능] F-1: `acceptTaskFromWeb()` return에 `success: true` 누락

- **재현 단계**:
  1. 담당자가 수락대기 업무에서 "✅ 수락" 클릭
  2. `acceptTaskFromWeb()` 실행 → `{ message: "..." }` 반환
  3. 프론트엔드 `acceptTask()` (line 4273)에서 `if (res.success)` 체크
  4. `res.success`가 `undefined` → falsy → 에러 분기 진입
- **기대 결과**: 수락 성공 토스트 표시 + 업무 목록 갱신
- **실제 결과**: 에러 토스트 표시 (실제 DB는 변경됨 — 데이터 불일치 위험)
- **심각도**: MINOR (but 실질 영향 MAJOR급)
- **감점**: -5
- **수정 방안**: return 객체에 `success: true` 추가

```javascript
// 현재 (line 701):
return { message: userName + "님이 업무를 수락하고 시작했습니다." };

// 수정 필요:
return { success: true, message: userName + "님이 업무를 수락하고 시작했습니다." };
```

#### [MINOR/기능] F-2: `acceptTaskFromWeb()` 에러 return에도 `success` 미포함

- **위치**: line 667
- **현재**: `return { message: "수락대기 상태가 아닙니다. 현재: " + currentStatus };`
- **수정 필요**: `return { success: false, message: ... };`
- **감점**: -3
