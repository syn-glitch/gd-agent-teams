## 3. 시스템 개요

기존 캐시플로우 봇(Code.gs v17.0.0)을 확장하여 **3단계 사이클** 추가:

```
PHASE 1 (지출 계획 등록)
  → 워크스페이스 웹 페이지에서 등록 + Slack /지출계획 간편 등록 병행
  → 네이버 쇼핑 API 자동 최저가 검색 (MVP 포함)
  → #관리자_지출관리 채널 노티

PHASE 2 (실행 관리)
  → GAS 트리거로 D-3, D-1, D-Day, D+3 자동 알림
  → Slack DM + #관리자_지출관리 요약

PHASE 3 (사후 분석)
  → 실제 지출 등록 시 계획과 매칭
  → Claude AI 차이 분석 → 보고서 생성
  → 월말 종합 분석 (웹 대시보드 + Slack 보고)
```
