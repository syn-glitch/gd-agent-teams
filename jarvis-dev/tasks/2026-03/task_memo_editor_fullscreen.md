## 지시 원문

> 메모 등록 이후에 수정 버튼을 누르면 칸이 작고 작은 칸안에서 스크롤해서 작성해야함.
> 메모가 길어질 경우 에디터 환경이 너무 불편함.
> 메모가 넓은 에디터 화면에서 문서를 수정할 수 있도록 해줘.
> 마크다운 포맷 툴바 모달 안에 포함, 실수 방지를 위해 X/취소 버튼만.

## 에이전트 이해 요약

- **핵심 요청**: 메모 수정 시 인라인 textarea → 풀스크린 모달 에디터로 교체
- **작업 범위**: 프론트엔드(HTML/CSS/JS)만 수정, 백엔드(`editArchivedMemo()`) 변경 없음
- **완료 기준**: 수정 버튼 클릭 → 풀스크린 모달에서 마크다운 편집 → 저장 성공
