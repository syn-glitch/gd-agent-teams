## 지시 원문

> 직원들이 출근해서 주디워크스페이스 내 채팅 창에 출근이라고 입력해서 업로드하면 주디가 인사와 당일 업무를 요약해서 보여주는걸 만들면 좋겠어

## 에이전트 이해 요약

- **핵심 요청**: 주디 채팅창에 "출근" 입력 시 인사 + 당일 업무 브리핑 제공
- **작업 범위**: 프론트엔드 키워드 분기 + 백엔드 전용 함수 (옵션 A, AI 미사용)
- **완료 기준**: "출근" 입력 → 1~2초 내 본인 당일 업무 + 일정 브리핑 표시

## 확인된 요구사항

1. 키워드: "출근"만 (추후 확장 예정)
2. 업무 범위:
   - 일반 사용자: 본인 담당 업무만 표시
   - 관리자 ("송용남", "정혜림", "김관수"): 전체 팀원 업무도 표시
3. 구현 방식: 옵션 A (프론트 분기 + 백엔드 직접 조회, AI API 미사용)
