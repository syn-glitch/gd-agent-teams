## 1. 배경 및 문제 정의

### 1.1 현재 상태

메모 등록 후 "수정" 버튼을 클릭하면, 메모 카드 **내부**에 인라인 textarea가 삽입된다.

| 항목 | 현재 구현 | 문제 |
|------|----------|------|
| 에디터 위치 | 메모 카드 내부 (인라인) | 카드 영역에 갇혀 공간 부족 |
| 높이 | `min-height: 100px`, 수동 리사이즈 | 긴 메모 작성 시 스크롤 불가피 |
| 너비 | 카드 width 종속 | 사이드바 열림 시 더 좁아짐 |
| 포맷 툴바 | 없음 (순수 텍스트) | 마크다운 포맷 지원 부재 |

### 1.2 사용자 피드백 (대표)

> "메모 등록 이후에 수정 버튼을 누르면 칸이 작고 작은 칸안에서 스크롤해서 작성해야함.
> 메모가 길어질 경우 에디터 환경이 너무 불편함."

### 1.3 UX 벤치마크

| 서비스 | 편집 패턴 | 장점 |
|--------|----------|------|
| **Notion** | 풀스크린 모달 | 넓은 편집 영역, 집중 모드 |
| **Google Keep** | 카드 확장 모달 | 원본 위치에서 자연스럽게 확대 |
| **Slack 스레드** | 슬라이드 업 | 모바일 친화적 |
| **GitHub Issue** | 인라인 확장 + 탭 전환 | 미리보기 병행 |

**선정 패턴**: Notion 스타일 **풀스크린 모달 에디터**
- 공간 최대화, 긴 메모도 쾌적, 모바일 호환, 구현 난이도 중간

## 2. 기능 요구사항

### 2.1 필수 기능 (Must-Have)

| ID | 요구사항 | 설명 |
|----|---------|------|
| **F-1** | 풀스크린 모달 오픈 | 수정 버튼 클릭 → 화면 전체를 덮는 모달 에디터 표시 |
| **F-2** | 메타 정보 표시 | 모달 상단에 날짜 + 시간 표시 (어떤 메모인지 식별) |
| **F-3** | 넓은 textarea | 화면 높이 60% 이상, 내용에 따라 자동 확장 |
| **F-4** | 마크다운 포맷 툴바 | 기존 노트 작성용 툴바(B, I, S, H1~H3, 리스트, 코드블록, 링크)를 모달 내 포함 |
| **F-5** | 하단 고정 액션바 | 스크롤과 무관하게 취소/저장 버튼 항상 하단 고정 |
| **F-6** | 닫기 방식 | X 버튼 또는 취소 버튼만 (실수 방지 — 딤 영역 클릭 시 닫히지 않음) |
| **F-7** | 저장 연동 | 기존 `editArchivedMemo()` 백엔드 로직 그대로 사용 |
| **F-8** | ESC 키 닫기 | 키보드 사용자 편의 (내용 변경 시 확인 다이얼로그) |

### 2.2 선택 기능 (Nice-to-Have)

| ID | 요구사항 | 설명 |
|----|---------|------|
| N-1 | 미리보기 탭 | 마크다운 렌더링 미리보기 (Write/Preview 탭 전환) |
| N-2 | 자동 임시 저장 | 30초마다 localStorage에 임시 저장 |
| N-3 | 글자 수 카운터 | 현재 글자 수 / 최대 글자 수 표시 |

### 3.3 모달 사이즈 스펙

| 속성 | 데스크톱 | 모바일 |
|------|---------|--------|
| **너비** | `min(90vw, 800px)` | `100vw` (전체) |
| **높이** | `min(90vh, 900px)` | `100vh` (전체) |
| **textarea 높이** | `min-height: 60vh` | `min-height: 50vh` |
| **모달 위치** | 화면 중앙 정렬 | 전체 화면 |
| **테두리** | `border-radius: 12px` | `border-radius: 0` |
| **딤 배경** | `rgba(0,0,0,0.5)` | 없음 (전체 화면) |

### 3.4 인터랙션 상세

| 동작 | 트리거 | 결과 |
|------|--------|------|
| 모달 열기 | 수정(✏️) 버튼 클릭 | 딤 + 모달 페이드인 (0.2s) |
| 모달 닫기 — 저장 | 저장 버튼 클릭 | `editArchivedMemo()` 호출 → 성공 시 모달 닫기 + 토스트 |
| 모달 닫기 — 취소 | 취소 또는 ✕ 클릭 | 내용 변경 없으면 즉시 닫기, 변경 있으면 "수정 내용을 버리시겠습니까?" 확인 |
| 모달 닫기 — ESC | ESC 키 | 취소 버튼과 동일 동작 |
| 딤 영역 클릭 | 모달 외부 클릭 | **아무 동작 없음** (실수 방지) |
| 저장 중 상태 | 저장 버튼 클릭 후 | 버튼 비활성화 + "저장 중..." 텍스트 |
| 포맷 툴바 | B/I/S/H1~H3 등 클릭 | 선택 영역에 마크다운 문법 삽입 |

### 3.5 마크다운 포맷 툴바 스펙

기존 노트 작성 화면의 툴바를 동일하게 재사용한다.

| 버튼 | 기능 | 삽입 문법 |
|------|------|----------|
| **B** | 굵게 | `**텍스트**` |
| *I* | 기울임 | `*텍스트*` |
| ~~S~~ | 취소선 | `~~텍스트~~` |
| H1 | 제목 1 | `# 텍스트` |
| H2 | 제목 2 | `## 텍스트` |
| H3 | 제목 3 | `### 텍스트` |
| • | 순서 없는 목록 | `- 텍스트` |
| □ | 체크박스 | `- [ ] 텍스트` |
| « | 인용 | `> 텍스트` |
| {} | 코드 블록 | `` `코드` `` |
| 🔗 | 링크 | `[텍스트](URL)` |
